# Machine Learning Lab Work

This repository contains implementations of various machine learning and data analysis algorithms using Python and Jupyter Notebook.

## Algorithms Implemented

* K-Means Clustering
* Agglomerative Clustering
* Apriori Algorithm
* Decision Tree
* Linear Regression
* Non-Linear Regression
* Principal Component Analysis (PCA)
* Singular Value Decomposition (SVD)

## Tools Used

* Python
* NumPy
* Pandas
* Matplotlib
* Scikit-learn
* Jupyter Notebook

These implementations were developed as part of my machine learning laboratory coursework while learning fundamental data science concepts.
